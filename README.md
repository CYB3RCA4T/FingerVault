# FingerVault

## FingerVault is a project which focuses on using fingerprint devices as encryption keys using the fingerprint.

FingerVault allows you to encrypt and decrypt files or folders securely using a fingerprint scanner. The key is protected and will be stored in TPM, making your data accessible **only with your verified fingerprint**.

---

## Features

- Encrypt files with a **your fingerprint**
- Decrypt files only when the authorized fingerprint is verified
- Prevent double encryption or decryption of the same file
- Protect system directories from accidental encryption
- Simple and intuitive GUI for selecting files and managing operations
